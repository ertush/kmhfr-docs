`GET /api/facilities/job_titles/`

*Lists and creates job titles*

Created --  Date the record was Created
Updated -- Date the record was Updated
Created_by -- User who created the record
Updated_by -- User who updated the record
active  -- Boolean is the record active
deleted -- Boolean is the record deleted

<h3 id="api_facilities_job_titles_list-parameters">Parameters</h3>

|Name|In|Type|Required|Description|
|---|---|---|---|---|
|ordering|query|string|false|Which field to use when ordering the results.|
|page|query|integer|false|A page number within the paginated result set.|
|page_size|query|integer|false|Number of results to return per page.|

> Example responses

> 200 Response

<BrowserWindow>
```json
{
  "count": 0,
  "next": "http://example.com",
  "previous": "http://example.com",
  "results": [
    {
      "id": "497f6eca-6276-4993-bfeb-53cbbbba6f08",
      "name": "string",
      "abbreviation": "string",
      "description": "string",
      "search": "string",
      "created": "2019-08-24T14:15:22Z",
      "updated": "2019-08-24T14:15:22Z"
    }
  ]
}
```
</BrowserWindow>
